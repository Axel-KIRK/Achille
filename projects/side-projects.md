# Side projects (Couche 2)

## Stratégie "small bets"
Build-in-public, 3 micro-tools gratuits d'abord, puis monétisation progressive.

## Micro-tools (Phase 1)
1. **ScreenShot.style** — Styled code screenshots. ~4h build, zero backend.
2. **Readme.new** — README generation. ~4h build, zero backend.
3. **Palette.cool** — Image-to-Tailwind color extraction. ~4h build, zero backend.

Statut : planifié, non démarré.

## StackDigest (Phase 2 — long terme)
Digest AI hebdo personnalisé par stack technique.
Architecture : GitHub Actions + Supabase + Resend + Claude Haiku + X API.
Coût MVP : ~0€.
Business model : free → sponsoring à 1K users → freemium à 5K+.
Vision doc créé en mars 2026.
